## Fixed clean Linux native packaging

- **WHAT:** Patched the three pinned `llama-cpp-sys-2` build-script hard-link guards to test path entries without following symlinks, and advanced the content-bound package identity to `vulkan-infinity-v3`.
- **WHY:** `Path::exists()` follows dangling SONAME symlinks, reports false, and lets a duplicate library basename call `hard_link` into an already occupied path.
- **HOW:** The deterministic vendor patch now changes exactly three `!dst.exists()` guards to `symlink_metadata(&dst).is_err()`, updates Cargo's vendored checksum, and includes the build-script bytes in the native patch digest.
- **IMPACT:** Clean Linux builds no longer panic after successful Vulkan compilation; duplicate native library assets preserve first-wins behavior while staging still copies from the authoritative CMake install tree.
