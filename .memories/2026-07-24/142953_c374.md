## Made RC4 native packaging reproducible

- **WHAT:** Reworked package builds to use a deterministic Cargo-vendored dependency tree, patch all three pinned llama.cpp Vulkan division-by-zero infinity expressions, update vendored checksums, and bind a content-derived patch identity into both binaries and the package manifest.
- **WHY:** The Windows Vulkan archive was not reproducible because shader compilation treated undefined `1.0 / 0.0` constant expressions inconsistently.
- **HOW:** Added strict byte-level patching for `topk_moe.comp`, `copy_to_quant.comp`, and `flash_attn_split_k_reduce.comp`; added a `verify-patched` package gate; made Bash and PowerShell validate the exact pinned identity; hardened malformed checksum handling; added cross-platform and invalid-shape tests.
- **IMPACT:** RC4 packaging now fails closed if the native patch is absent or malformed, records the exact patched-source digest, and produces byte-identical Apple arm64 archives across repeated clean package runs.
